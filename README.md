# Calculator Program

## Overview
This Python program performs four fundamental arithmetic operations: addition, subtraction, multiplication, and division. It accepts two numeric values from the user and displays the calculated results.

## Functional Requirements
- Accept two user inputs
- Convert values into numeric format
- Perform arithmetic operations
- Display outputs clearly

## Software Design Principles Applied

### 1. Readability
The program uses descriptive variable names such as `num1` and `num2`, making the code easy to understand.

### 2. Simplicity
The solution follows a straightforward structure without unnecessary complexity, which improves usability for beginner programmers.

### 3. Maintainability
Each operation is clearly separated in output statements, making future edits simple (for example, adding modulus or exponent functions).

### 4. Reusability
The logic can be moved into functions such as `add()`, `subtract()`, and reused in larger systems.

### 5. Input Handling
Using `float()` allows decimal values, increasing flexibility over integer-only input.

## Possible Improvements
- Add error handling for division by zero
- Use functions for each operation
- Create menu-driven interface
